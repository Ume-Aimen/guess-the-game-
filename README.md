# 🎮 Guess the Number Game

A simple and interactive number guessing game built using HTML, CSS, and JavaScript.  
This project is perfect for beginners who want to practice basic JavaScript concepts and DOM manipulation.

---

## 🕹 Game Rules

- The computer randomly generates a number.
- **Level 1:** The number is between **1 and 10**.
- The player has **3 attempts** to guess the correct number.
- If the guess is lower than the actual number, a **"Too Low!"** message appears.
- If the guess is higher than the actual number, a **"Too High!"** message appears.
- Invalid inputs (empty, non-numeric, or out-of-range values) are ignored and **do not count as attempts**.
- If the player guesses correctly within 3 attempts, they advance to **Level 2**.
- **Level 2:** The number range increases to **1 to 25**, with again only **3 attempts**.
- If the player fails to guess the number within the allowed attempts at any level, the game ends.
- A **Restart button** appears after winning or losing the game.
